# MC Control v2

Minecraft server panel for Termux (Android). Aternos-style web UI.

## Install

Paste this in Termux:
```bash
curl -fsSL https://raw.githubusercontent.com/loaf1ms/mc-control/main/setup.sh | bash
```

Then start it:
```bash
~/start-mc.sh        # foreground (shows live logs)
~/start-mc-bg.sh     # background via tmux
```

Open in browser: `http://your_phones_ip:8080`
